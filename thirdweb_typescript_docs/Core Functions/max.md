Returns the maximum of two BigInt values.

## Example

`max(1n,2n);// 2n`
#### Signature

`functionmax(a:bigint,b:bigint):bigint;`
## Parameters

#### a

The first BigInt value.

### Type

`leta:bigint;`
#### b

The second BigInt value.

### Type

`letb:bigint;`
## Returns

#### Return Type

`letreturnType:bigint;`The larger of the two BigInt values.
